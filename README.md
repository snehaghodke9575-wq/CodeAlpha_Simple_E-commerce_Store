This is a fully responsive E-Commerce web application built to simulate a real-world online shopping experience. The platform allows users to browse products, 
add items to the cart, manage quantities, and proceed to checkout.
The project focuses on building core e-commerce functionalities including product listing, shopping cart management, dynamic UI updates, 
and user-friendly navigation.
Features
1.Product listing page
2.Add to cart functionality
3.Remove items from cart
4.Quantity management
5.Responsive design
6.Clean and user-friendly UI
Tech Stack
HTML
CSS
JavaScript
Python
